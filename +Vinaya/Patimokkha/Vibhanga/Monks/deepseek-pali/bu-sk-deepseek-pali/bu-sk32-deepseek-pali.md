# BU-SK 32

## 606. Tena samayena buddho bhagavā sāvatthiyaṃ viharati jetavane anāthapiṇḍikassa ārāme. Tena kho pana samayena chabbaggiyā bhikkhū asakkaccaṃ piṇḍapātaṃ bhuñjanti abhuñjitukāmā viya…pe….

## 606.At that time the Blessed One was dwelling at Sāvatthī in Jeta Grove, Anāthapiṇḍika’s Monastery. Now at that time the group of six bhikkhus were eating almsfood disrespectfully, as if not wanting to eat….

‘‘Sakkaccaṃ piṇḍapātaṃ bhuñjissāmīti sikkhā karaṇīyā’’ti.

‘I will eat almsfood respectfully’—this is a training to be observed.

Sakkaccaṃ piṇḍapāto bhuñjitabbo. Yo anādariyaṃ paṭicca asakkaccaṃ piṇḍapātaṃ bhuñjati, āpatti dukkaṭassa.

Almsfood should be eaten respectfully. Whoever, being disrespectful, eats almsfood disrespectfully, commits an offense of wrongdoing.

Anāpatti asañcicca, assatiyā, ajānantassa, gilānassa, āpadāsu, ummattakassa,

There is no offense if unintentional, if unmindful, if unknowing, if sick, in times of danger, if insane,

Ādikammikassāti.

for the first offender.

Paṭhamasikkhāpadaṃ niṭṭhitaṃ.

The first training rule is completed.
