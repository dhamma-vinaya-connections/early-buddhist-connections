# BU-SK 28

## 602. Tena samayena buddho bhagavā sāvatthiyaṃ viharati jetavane anāthapiṇḍikassa ārāme. Tena kho pana samayena chabbaggiyā bhikkhū asakkaccaṃ piṇḍapātaṃ paṭiggaṇhanti chaḍḍetukāmā viya…pe….

## 602.At that time the Blessed One was dwelling at Savatthi, in Jeta Grove, Anathapindika's Monastery. Now at that time the group-of-six bhikkhus received almsfood carelessly, as if wishing to discard it...and so on.

‘‘Sakkaccaṃ piṇḍapātaṃ paṭiggahessāmīti sikkhā karaṇīyā’’ti.

“I will respectfully receive almsfood”—this is a training to be observed.

Sakkaccaṃ piṇḍapāto paṭiggahetabbo. Yo anādariyaṃ paṭicca asakkaccaṃ piṇḍapātaṃ paṭiggaṇhāti chaḍḍetukāmo viya, āpatti dukkaṭassa.

Almsfood should be received respectfully. Whoever, being disrespectful, receives almsfood irreverently, as if wishing to discard it, commits an offense of wrong conduct.

Anāpatti asañcicca…pe… ādikammikassāti.

There is no offense if it is unintentional...or if one is the first offender.

Sattamasikkhāpadaṃ niṭṭhitaṃ.

The seventh training rule is concluded.
