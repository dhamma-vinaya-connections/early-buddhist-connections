# Nuns - Pacittiya - Confession
## 7 Gabbhinivaggo - 7 Pregnant Woman
## BNI-PC61 - Gabbhinī - Ordaining a Pregnant Woman

### 🌐Overview

| Rule in Pali  | Rule in English  |
| :-----------: | :--------------: |
| Gabbhinī: Yā pana bhikkhunī gabbhiniṃ vuṭṭhāpeyya, pācittiyaṃ. | 61. Pregnant Ordination: Should any bhikkhunī ordain a pregnant woman... it is to be confessed. |
