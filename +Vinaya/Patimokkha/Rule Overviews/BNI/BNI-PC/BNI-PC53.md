# Nuns - Pacittiya - Confession

## BNI-PC53 - Gaṇaparibhāsana - Reviling a Group

### 🌐Overview

| Rule in Pali  | Rule in English  |
| :-----------: | :--------------: |
| Caṇḍīkatā: Yā pana bhikkhunī caṇḍīkatā gaṇassa paribhāsikā, pācittiyaṃ. | 53. Gossip: Should any bhikkhunī, having a hot temper, gossip about a group... it is to be confessed. |
