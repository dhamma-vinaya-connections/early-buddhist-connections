# Nuns - Pacittiya - Confession

## BNI-PC47 - Āvasathacīvara - Monastery Robes

### 🌐Overview

| Rule in Pali  | Rule in English  |
| :-----------: | :--------------: |
| Āvasathacīvara: Yā pana bhikkhunī āvasathacīvaraṃ na paṭinissajjeyya, pācittiyaṃ. | 47. Menstruation Cloth: Should any bhikkhunī use a menstruation cloth without washing it (or giving it to be washed)... it is to be confessed. |
