# Nuns - Pacittiya - Confession
## 10 Musāvāda-Vaggo - 10 Lies
## BNI-PC106 - Pathavīkhaṇana - Digging Earth

### 🌐Overview

| Rule in Pali  | Rule in English  |
| :-----------: | :--------------: |
| Pathavikhaṇana: Yā pana bhikkhunī pathaviṁ khaṇeyya vā khaṇāpeyya vā, pācittiyaṁ. | 106. Digging Soil: Should any bhikkhunī dig soil or have it dug, it is to be confessed. |


> [!Info] Analisis of the rule
> 
> >[!info]- Offence Factors 
> > 
> > > From BMC1
> > 
> > 1 Object. The Pali word for soil, paṭhavī, also means ground or earth. 2 Effort. The Vibhaṅga says that the term digging also covers burning, 3 Perception. If one perceives a patch of genuine soil not to be genuine soil, it is not grounds for an offense. If one is in doubt as to whether a patch of earth is classed as genuine soil, it is grounds for a dukkaṭa regardless of whether it is actually classed as genuine soil or not. 4 Intention to make a hole in the ground
> 
> > [!info]- No Offence 
> > > From Vibhanga
> > 
> > If he speaks, saying: “Find this, give this, convey this, this is wanted, make this allowable”; if it was unintentional, if (he was) not thinking, if he did not know, if he is mad, if he is the first wrong-doer
> 
> > [!info]- Useful Notes
> > > From various authors
> > 
> > Kappiya-vohāra—“allowable expression,” or “wording it right” use in the context of rules where an express command would be an offense, but an indication of a desire or intent would not. This rule, together with the following one, also effectively prevents bhikkhus from engaging in agriculture
>
