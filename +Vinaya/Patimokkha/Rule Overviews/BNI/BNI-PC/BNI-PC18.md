# Nuns - Pacittiya - Confession
## 2 Rattandhakāravaggo - 2 Darkness
## BNI-PC18 - Paraujjhāpanaka - Complaining About Others

### 🌐Overview

| Rule in Pali  | Rule in English  |
| :-----------: | :--------------: |
| Anokāsa: Yā pana bhikkhunī... akkataññū aññāya vā, pācittiyaṃ. | 18. Misunderstanding: Should any bhikkhunī teach the Dhamma without analyzing the meaning/text... it is to be confessed. |
