# Nuns - Pacittiya - Confession
## 5 Cittāgāravaggo - 5 Picture gallery
## BNI-PC49 - Tiracchānavijjāpariyāpuṇana - Learning Pointless Knowledge

### 🌐Overview

| Rule in Pali  | Rule in English  |
| :-----------: | :--------------: |
| Tiracchānavijjā: Yā pana bhikkhunī tiracchānavijjaṃ pariyāpuṇeyya, pācittiyaṃ. | 49. Learning Worldly Arts: Should any bhikkhunī learn worldly arts (fortune telling, spells)... it is to be confessed. |


> [!Info] Analisis of the rule
> 
> >[!info]- Offence Factors 
> > 
> > > From BMC1
> > 
> > Worldly subjects: whatever is external to the Buddha’s Teaching, not connected with the goal
> 
> > [!info]- No Offence 
> > > From Vibhanga
> > 
> > If she learns writing; if she learns protective verses; if she learns verses for the purpose of protection; if she is insane; if she is the first offender
> 
> > [!info]- Useful Notes
> > > From various authors
> > 
> > Applicable to Bhikkhu as dukkata offences
>
