# Nuns - Nissaggiya Pacittiya - Relinquish with confession
## Pattavagga - Bowl
## BNI-NP4 - Aññaviññāpana - Asking for Something Else

### 🌐Overview

| Rule in Pali  | Rule in English  |
| :-----------: | :--------------: |
| Aññadatthika: Yā pana bhikkhunī aññadatthikena parikkhārena aññuddisikena mahājanikena saññācikena aññaṃ cetāpeyya, nissaggiyaṃ pācittiyaṃ. | 4. Misusing Requisites (I): Should any bhikkhunī use a requisite (gift) asking for it for one purpose but using it for another... it is to be forfeited and confessed. |
