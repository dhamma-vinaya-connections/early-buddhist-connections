# Nuns - Nissaggiya Pacittiya - Relinquish with confession
## Pattavagga - Bowl
## BNI-NP8 - Gaṇikacetāpana - Exchanging Group Property

### 🌐Overview

| Rule in Pali  | Rule in English  |
| :-----------: | :--------------: |
| Dutiya-aññaviññāpana: Yā pana bhikkhunī aññātakaṃ gahapatiṃ vā... cīvaraṃ viññāpeyya... nissaggiyaṃ pācittiyaṃ. | 8. Unrelated Donors (II): Should a bhikkhunī ask for a robe from an unrelated householder (who intended to give something else)... it is to be forfeited and confessed. |


> [!Info] Analisis of the rule
> 
> >[!info]- Offence Factors 
> > 
> > > From BMC1
> > 
> > Collective: belonging to a group, not to the Sangha, not to an individual nun
> 
> > [!info]- No Offence 
> > > From Vibhanga
> > 
> > If she uses the remainder; if she uses it after getting permission from the owners; if there is an emergency; if she is insane; if she is the first offender
> 
>
