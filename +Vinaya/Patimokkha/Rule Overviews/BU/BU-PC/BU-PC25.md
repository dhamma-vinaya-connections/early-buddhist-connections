# bhikkhus - Pacittiya - Confession
## 3 Bhikkhunovāda-Vaggo - 3 Instruction
## BU-PC25 - Cīvaradāna - Robes Giving

### 🌐Overview

| Rule in Pali  | Rule in English  |
| :-----------: | :--------------: |
| Yo pana bhikkhu aññātikāya bhikkhuniyā cīvaraṃ dadeyya aññatra pārivattakā, pācittiyaṃ | If a monk gives robe-cloth to an unrelated nun, except in exchange, he commits an offense entailing confession |


> [!Info] Analisis of the rule
> 
> >[!info]- Offence Factors 
> > 
> > > From BMC1
> > 
> > 1) Object: robe-cloth measuring at least 4 x 8 fingerbreadths. 2) Effort: bhikkhu gives the cloth to an unrelated bhikkhunī and does not receive anything from her in exchange
> 
> > [!info]- No Offence 
> > > From Vibhanga
> > 
> > If she is a relation; if there is an exchange; if there is a large thing for a small thing, or vice versa ; if a nun takes it on trust; if she takes it for the time being; if he gives another requisite, except robe-material; if she is a female probationer, a female novice; if he is mad, if he is the first wrong-doer
> 
> > [!info]- Useful Notes
> > > From various authors
> > 
> > The item given in exchange for the cloth, can be worth much more or much less
> 
>
