18. "Monks, if there were no gratification in forms, beings would not be enamored with forms. But because there is gratification in forms, beings are enamored with forms. If there were no danger in forms, beings would not grow disenchanted with forms. But because there is danger in forms, beings grow disenchanted with forms. If there were no escape from forms, beings would not escape from forms. But because there is an escape from forms, beings escape from forms. If there were no gratification in sounds... aromas... flavors... tactile sensations... mental phenomena, beings would not be enamored with mental phenomena. But because there is gratification in mental phenomena, beings are enamored with mental phenomena. If there were no danger in mental phenomena, beings would not grow disenchanted with mental phenomena. But because there is danger in mental phenomena, beings grow disenchanted with mental phenomena. If there were no escape from mental phenomena, beings would not escape from mental phenomena. But because there is an escape from mental phenomena, beings escape from mental phenomena."
