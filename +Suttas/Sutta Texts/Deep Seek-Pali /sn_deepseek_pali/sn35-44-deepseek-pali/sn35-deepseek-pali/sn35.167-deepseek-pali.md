# SN 35.167 Khīrarukkhopamasuttaṃ

## SN 35.167 The Simile of the Milk-tree

231. ‘‘Yassa kassaci, bhikkhave, bhikkhussa vā bhikkhuniyā vā cakkhuviññeyyesu rūpesu yo rāgo so atthi, yo doso so atthi, yo moho so atthi, yo rāgo so appahīno, yo doso so appahīno, yo moho so appahīno tassa parittā cepi cakkhuviññeyyā rūpā cakkhussa āpāthaṃ āgacchanti pariyādiyantevassa cittaṃ; ko pana vādo adhimattānaṃ! Taṃ kissa hetu? Yo, bhikkhave, rāgo, so atthi, yo doso so atthi, yo moho so atthi, yo rāgo so appahīno, yo doso so appahīno, yo moho so appahīno…pe….

231.“bhikkhus, for any monk or nun in whom there is passion, aversion, and delusion regarding forms cognizable by the eye—whose passion is not abandoned, whose aversion is not abandoned, whose delusion is not abandoned—even if only a few forms cognizable by the eye come into the range of vision, they overwhelm his mind; how much more so when they are excessive! Why is that? Because, bhikkhus, there is passion, aversion, and delusion—whose passion is not abandoned, whose aversion is not abandoned, whose delusion is not abandoned… and so on.”

‘‘Yassa kassaci, bhikkhave, bhikkhussa vā bhikkhuniyā vā jivhāviññeyyesu rasesu yo rāgo so atthi…pe….

“bhikkhus, for any monk or nun in whom there is passion regarding tastes cognizable by the tongue… and so on.”

‘‘Yassa kassaci, bhikkhave, bhikkhussa vā bhikkhuniyā vā manoviññeyyesu dhammesu yo rāgo so atthi, yo doso so atthi, yo moho so atthi, yo rāgo so appahīno, yo doso so appahīno, yo moho so appahīno, tassa parittā cepi manoviññeyyā dhammā manassa āpāthaṃ āgacchanti pariyādiyantevassa cittaṃ; ko pana vādo adhimattānaṃ! Taṃ kissa hetu? Yo, bhikkhave, rāgo, so atthi, yo doso so atthi, yo moho so atthi, yo rāgo so appahīno, yo doso so appahīno, yo moho so appahīno.

“bhikkhus, for any monk or nun in whom there is passion, aversion, and delusion regarding mind-cognizable phenomena—whose passion is not abandoned, whose aversion is not abandoned, whose delusion is not abandoned—even if only a few mind-cognizable phenomena come into the range of the mind, they overwhelm his mind; how much more so when they are excessive! Why is that? Because, bhikkhus, there is passion, aversion, and delusion—whose passion is not abandoned, whose aversion is not abandoned, whose delusion is not abandoned.”

‘‘Seyyathāpi, bhikkhave, khīrarukkho assattho vā nigrodho vā pilakkho vā udumbaro vā daharo taruṇo komārako. Tamenaṃ puriso tiṇhāya kuṭhāriyā yato yato ābhindeyya [bhindeyya (syā. kaṃ. sī. aṭṭha.), abhindeyya (katthaci)] āgaccheyya khīra’’nti? ‘‘Evaṃ, bhante’’. ‘‘Taṃ kissa hetu’’? ‘‘Yañhi, bhante, khīraṃ taṃ atthī’’ti.

“Suppose, bhikkhus, there is a milk-tree—an assattha tree, a banyan tree, a pilakkha tree, or a fig tree—young, tender, a sapling. If a man were to strike it with a sharp axe here and there, would sap come out?” “Yes, venerable sir.” “Why is that?” “Because, venerable sir, sap exists in it.”

‘‘Evameva kho, bhikkhave, yassa kassaci bhikkhussa vā bhikkhuniyā vā cakkhuviññeyyesu rūpesu yo rāgo so atthi, yo doso so atthi, yo moho so atthi, yo rāgo so appahīno, yo doso so appahīno, yo moho so appahīno, tassa parittā cepi cakkhuviññeyyā rūpā cakkhussa āpāthaṃ āgacchanti pariyādiyantevassa cittaṃ; ko pana vādo adhimattānaṃ! Taṃ kissa hetu? Yo, bhikkhave, rāgo so atthi, yo doso so atthi, yo moho so atthi, yo rāgo so appahīno, yo doso so appahīno, yo moho so appahīno…pe….

“In the same way, bhikkhus, for any monk or nun in whom there is passion, aversion, and delusion regarding forms cognizable by the eye—whose passion is not abandoned, whose aversion is not abandoned, whose delusion is not abandoned—even if only a few forms cognizable by the eye come into the range of vision, they overwhelm his mind; how much more so when they are excessive! Why is that? Because, bhikkhus, there is passion, aversion, and delusion—whose passion is not abandoned, whose aversion is not abandoned, whose delusion is not abandoned… and so on.”

‘‘Yassa kassaci, bhikkhave, bhikkhussa vā bhikkhuniyā vā jivhāviññeyyesu rasesu yo rāgo so atthi…pe….

“bhikkhus, for any monk or nun in whom there is passion regarding tastes cognizable by the tongue… and so on.”

‘‘Yassa kassaci, bhikkhave, bhikkhussa vā bhikkhuniyā vā manoviññeyyesu dhammesu yo rāgo so atthi, yo doso so atthi, yo moho so atthi, yo rāgo so appahīno, yo doso so appahīno, yo moho so appahīno, tassa parittā cepi manoviññeyyā dhammā manassa āpāthaṃ āgacchanti pariyādiyantevassa cittaṃ; ko pana vādo adhimattānaṃ! Taṃ kissa hetu? Yo, bhikkhave, rāgo so atthi, yo doso so atthi, yo moho so atthi, yo rāgo so appahīno, yo doso so appahīno, yo moho so appahīno.

“bhikkhus, for any monk or nun in whom there is passion, aversion, and delusion regarding mind-cognizable phenomena—whose passion is not abandoned, whose aversion is not abandoned, whose delusion is not abandoned—even if only a few mind-cognizable phenomena come into the range of the mind, they overwhelm his mind; how much more so when they are excessive! Why is that? Because, bhikkhus, there is passion, aversion, and delusion—whose passion is not abandoned, whose aversion is not abandoned, whose delusion is not abandoned.”

‘‘Yassa kassaci, bhikkhave, bhikkhussa vā bhikkhuniyā vā cakkhuviññeyyesu rūpesu yo rāgo so natthi, yo doso so natthi, yo moho so natthi, yo rāgo so pahīno, yo doso so pahīno, yo moho so pahīno, tassa adhimattā cepi cakkhuviññeyyā rūpā cakkhussa āpāthaṃ āgacchanti nevassa cittaṃ pariyādiyanti; ko pana vādo parittānaṃ! Taṃ kissa hetu? Yo, bhikkhave, rāgo so natthi, yo doso so natthi, yo moho so natthi, yo rāgo so pahīno, yo doso so pahīno, yo moho so pahīno…pe….

“bhikkhus, for any monk or nun in whom there is no passion, no aversion, and no delusion regarding forms cognizable by the eye—whose passion is abandoned, whose aversion is abandoned, whose delusion is abandoned—even if excessive forms cognizable by the eye come into the range of vision, they do not overwhelm his mind; how much less so when they are few! Why is that? Because, bhikkhus, there is no passion, no aversion, and no delusion—whose passion is abandoned, whose aversion is abandoned, whose delusion is abandoned… and so on.”

‘‘Yassa kassaci, bhikkhave, bhikkhussa vā bhikkhuniyā vā jivhāviññeyyesu rasesu…pe… manoviññeyyesu dhammesu yo rāgo so natthi, yo doso so natthi, yo moho so natthi, yo rāgo so pahīno, yo doso so pahīno, yo moho so pahīno, tassa adhimattā cepi manoviññeyyā dhammā manassa āpāthaṃ āgacchanti nevassa cittaṃ pariyādiyanti; ko pana vādo parittānaṃ! Taṃ kissa hetu? Yo, bhikkhave, rāgo so natthi, yo doso so natthi, yo moho so natthi, yo rāgo so pahīno, yo doso so pahīno, yo moho so pahīno. Seyyathāpi, bhikkhave, khīrarukkho assattho vā nigrodho vā pilakkho vā udumbaro vā sukkho kolāpo terovassiko. Tamenaṃ puriso tiṇhāya kuṭhāriyā yato yato ābhindeyya āgaccheyya khīra’’nti? ‘‘No hetaṃ, bhante’’. ‘‘Taṃ kissa hetu’’? ‘‘Yañhi, bhante, khīraṃ taṃ natthī’’ti.

“bhikkhus, for any monk or nun in whom there is no passion, no aversion, and no delusion regarding tastes cognizable by the tongue… and regarding mind-cognizable phenomena—whose passion is abandoned, whose aversion is abandoned, whose delusion is abandoned—even if excessive mind-cognizable phenomena come into the range of the mind, they do not overwhelm his mind; how much less so when they are few! Why is that? Because, bhikkhus, there is no passion, no aversion, and no delusion—whose passion is abandoned, whose aversion is abandoned, whose delusion is abandoned. Just as, bhikkhus, a milk tree—whether an assattha, a nigrodha, a pilakkha, or an udumbara—is a dry stump that has been dead for a long time. If a man were to strike it here and there with a sharp axe, would sap come out?” “No, venerable sir.” “Why is that?” “Because, venerable sir, there is no sap in it.”

‘‘Evameva kho, bhikkhave, yassa kassaci bhikkhussa vā bhikkhuniyā vā cakkhuviññeyyesu rūpesu yo rāgo so natthi, yo doso so natthi, yo moho so natthi, yo rāgo so pahīno, yo doso so pahīno, yo moho so pahīno, tassa adhimattā cepi cakkhuviññeyyā rūpā cakkhussa āpāthaṃ āgacchanti nevassa cittaṃ pariyādiyanti; ko pana vādo parittānaṃ! Taṃ kissa hetu? Yo, bhikkhave, rāgo so natthi, yo doso so natthi, yo moho so natthi, yo rāgo so pahīno, yo doso so pahīno, yo moho so pahīno…pe….

“Just so, bhikkhus, for any monk or nun in whom there is no passion, no aversion, and no delusion regarding forms cognizable by the eye—whose passion is abandoned, whose aversion is abandoned, whose delusion is abandoned—even if excessive forms cognizable by the eye come into the range of vision, they do not overwhelm his mind; how much less so when they are few! Why is that? Because, bhikkhus, there is no passion, no aversion, and no delusion—whose passion is abandoned, whose aversion is abandoned, whose delusion is abandoned… and so on.”

‘‘Yassa kassaci, bhikkhave, bhikkhussa vā bhikkhuniyā vā jivhāviññeyyesu rasesu…pe….

“bhikkhus, for any monk or nun in whom there is no passion regarding tastes cognizable by the tongue… and so on.”

‘‘Yassa kassaci, bhikkhave, bhikkhussa vā bhikkhuniyā vā manoviññeyyesu dhammesu yo rāgo so natthi, yo doso so natthi, yo moho so natthi, yo rāgo so pahīno, yo doso so pahīno, yo moho so pahīno, tassa adhimattā cepi manoviññeyyā dhammā manassa āpāthaṃ āgacchanti, nevassa cittaṃ pariyādiyanti; ko pana vādo parittānaṃ! Taṃ kissa hetu? Yo, bhikkhave, rāgo so natthi, yo doso so natthi, yo moho so natthi, yo rāgo so pahīno, yo doso so pahīno, yo moho so pahīno’’ti. Catutthaṃ.

“bhikkhus, for any monk or nun in whom there is no passion, no aversion, and no delusion regarding mind-cognizable phenomena—whose passion is abandoned, whose aversion is abandoned, whose delusion is abandoned—even if excessive mind-cognizable phenomena come into the range of the mind, they do not overwhelm his mind; how much less so when they are few! Why is that? Because, bhikkhus, there is no passion, no aversion, and no delusion—whose passion is abandoned, whose aversion is abandoned, whose delusion is abandoned.” Fourth Sutta.
