## Saṁyutta Nikāya — The Linked Discourses

## 43. Asaṅkhatasaṁyutta: On the Unconditioned

### II. The Second Chapter — SN43.14-43: Undefiled, Etc.

2the truth …

3the far shore …

4the subtle …

5the very hard to see …

6the unaging …

7the constant …

8the not falling apart …


9the invisible …

10the unproliferated …

11the peaceful …


12the deathless …

13the sublime …

14the state of grace …

15the sanctuary …

16the ending of craving …

17the incredible …

18the amazing …

19the untroubled …

20the not liable to trouble …


21extinguishment …

22the unafflicted …

23dispassion …

24purity …

27the island …

28the protection …

29the shelter …
